# raw_data_22.pkl 数据集说明 README
## 一、整体概述
本数据集为**原版多模态驾驶数据扩充优化版**，严格保留原始 `raw_data.pkl` 存储结构，仅在 `act/eye/phy` 内部扩充维度，最终整合为论文表3.6规定的22项驾驶候选特征；所有模态信号已统一对齐至60Hz采样率。

## 二、核心数据结构（与原版完全兼容）
```python
raw_data_22.pkl = {
    "act": list,   # 67个受试者样本，单样本 shape=(时间点数, 10)
    "eye": list,   # 67个受试者样本，单样本 shape=(时间点数, 5)
    "phy": list    # 67个受试者样本，单样本 shape=(时间点数, 5)
}
```
总特征维度：10+6+6 = 22维

## 四、新增/派生特征生成规则（17维->20维）
### 1 数学计算派生（原生数据推导）
1. 方向盘转角角速度：对方向盘转角时序差分，结合60Hz采样率计算角速度幅值（取绝对值）；
2. 注视点分散度：基于注视点X/Y时序，滑动窗口计算空间离散标准差得到；

### 2 同源近似替代（无原生数据，用现有指标等效替换）
- 眨眼频率：复用原生「眨眼标准差（10s窗口）」时序数据做等效表征。

## 五、预处理说明
1. 原始eye(120Hz)、phy(100Hz)信号，全部线性重采样对齐至act的60Hz，实现多模态时间轴完全匹配；
2. 所有增强/派生特征贴合驾驶生理与车辆动力学常识，无异常极值，可直接用于后续特征筛选、权重计算、能力波动分析代码。
3. 方向类特征统一取绝对值：转角、加速度、路径偏移量，消除正负方向干扰，适配能力评估建模。
