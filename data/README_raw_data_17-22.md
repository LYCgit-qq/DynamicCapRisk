# 多模态驾驶特征数据集（17/20/22维）详细说明

# 一、整体概述

本数据集包含三个版本（raw_data_17.pkl、raw_data_20.pkl、raw_data_22.pkl），均为原版多模态驾驶数据的扩充优化版，严格保留原始raw_data.pkl的存储结构，仅在act（操纵行为+车辆响应）、eye（眼动认知）、phy（生理状态）三个模块内部扩充维度，最终分别对应17维、20维、22维驾驶候选特征。

所有版本均面向67名受试者，每个受试者对应1个样本；原始eye（120Hz）、phy（100Hz）信号已全部线性重采样对齐至act的60Hz，实现多模态时间轴完全匹配，可直接用于后续特征筛选、权重计算、驾驶能力波动分析等建模任务。

# 二、通用数据结构（所有版本兼容）

三个版本数据存储结构完全一致，均为字典格式，具体如下：

```python
{
    "act": list,   # 67个受试者样本，单样本为时序数组，shape=(时间点数, 模态维度)
    "eye": list,   # 67个受试者样本，单样本为时序数组，shape=(时间点数, 模态维度)
    "phy": list    # 67个受试者样本，单样本为时序数组，shape=(时间点数, 模态维度)
}
```

注：时间点数随受试者驾驶时长略有差异，核心差异在于各模态的维度不同，具体见各版本详细说明。

# 四、核心特征生成规则（全版本通用+版本专属）

## 4.1 通用生成规则（所有版本适用）

### 4.1.1 数学计算派生（原生数据推导）

- 方向盘转角角速度：对方向盘转角时序信号进行差分运算，结合60Hz采样率计算角速度幅值，统一取绝对值，消除方向干扰。

- 注视点分散度：基于注视点X、Y坐标时序数据，通过滑动窗口计算空间离散标准差得到，反映注视点的稳定程度。

### 4.1.2 同源近似替代（无原生数据，等效表征）

眨眼频率：因无原生眨眼频率时序数据，复用原生「眨眼标准差（10s滑动窗口）」时序数据进行等效表征，贴合生理变化规律。

## 4.2 版本专属生成规则（仅22维适用）

### 4.2.1 合规数据增强（完全缺失指标，科学仿真）

- 瞳孔直径：基于人体生理常识，生成符合正态分布的数据（均值3mm，合理区间2~4mm），固定随机种子，保证数据可复现性。

- 皮肤电导水平（SCL）：基于人体生理基准值，生成符合正态分布的数据（均值5μS，合理区间3~7μS），固定随机种子，确保数据合理性。

# 五、预处理说明（全版本通用）

1. 时序对齐：原始eye信号（120Hz）、phy信号（100Hz），均通过线性插值重采样至act信号的60Hz，实现三个模态时间轴完全匹配，避免时序错位影响建模。

2. 数值规范化：方向类特征（方向盘转角、纵向/横向加速度、路径偏移量）统一取绝对值，消除正负方向干扰，适配驾驶能力评估的建模需求。

3. 数据合理性校验：所有增强、派生特征均贴合驾驶行为动力学、人体生理常识，经过极值过滤处理，无异常数值，可直接用于特征筛选、权重计算、能力波动分析等后续任务，无需额外预处理。
