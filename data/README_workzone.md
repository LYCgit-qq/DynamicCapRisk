# 施工区数据处理工具

## 项目概述
本工具用于将施工区Excel数据转换为连续距离序列的CSV文件，支持自动填充车道数、限速等属性，并精准识别道路几何类型（straight/bend/cross）区间。

## 数据说明

### 输入数据
- **文件格式**：Excel (.xlsx)
- **子表要求**：需包含 `test00`、`test01`、`test02` 三个子表
- **表头字段**：距离 (m)、车道数、限速 (km/h)、施工区状态、标识牌类型、标识牌编码、车道变化预告、道路几何类型

### 标识牌编码说明
| 标识牌编码 | 标识牌类型       | 说明                     |
|------------|------------------|--------------------------|
| 0          | -                | 无标识牌                 |
| 1          | 60limit          | 限速60km/h标识牌         |
| 2          | 200m             | 200m预告标识牌           |
| 3          | 3->2             | 车道由3变2预告标识牌     |
| 4          | 40limit          | 限速40km/h标识牌         |
| 5          | 施工结束         | 施工区结束标识牌         |
| 6          | Cross（开始）    | 交叉路段开始标识牌       |
| 6          | Cross（结束）    | 交叉路段结束标识牌       |
| 7          | Bend（开始）     | 弯道路段开始标识牌       |
| 7          | Bend（中段）     | 弯道路段中段标识牌       |
| 7          | Bend（结束）     | 弯道路段结束标识牌       |
| 9          | Sign             | 通用指示标识牌           |
| 10         | 4->3             | 车道由4变3标识牌         |
| 11         | 2->4             | 车道由2变4标识牌         |



## 输出文件说明
- **文件数量**：3个CSV文件（`test00_continuous.csv`、`test01_continuous.csv`、`test02_continuous.csv`）
