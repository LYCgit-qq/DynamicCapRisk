# AHP打分方法简要说明
本次打分基于**AHP（层次分析法）1-9标度法**，结合驾驶能力波动评估的业务逻辑设计，核心规则如下：

#### 1. 基础打分规则（AHP标准）
- 1：两特征**同等重要**；3：稍重要；5：明显重要；7：强烈重要；9：极端重要（数值越大，前者相对后者越重要）；
- 仅填写矩阵右上三角（脚本自动补左下三角的倒数，满足“互反性”）；
- 最终校验CR<0.1（一致性合格），确保打分逻辑无矛盾。

#### 2. 分层优先级（核心逻辑）
按特征对驾驶能力波动的影响程度，从高到低分4层：
- **核心操纵层**（方向盘相关）：steering_angle/steering_velocity，打分6-9（驾驶控制核心，波动影响最大）；
- **车辆响应层**（制动/油门/偏移等）：brake_pedal/throttle_pedal/lateral_offset等，打分3-8（直接反映行驶稳定性）；
- **眼动认知层**（注视/眨眼）：gaze_dispersion/blink_frequency，打分2-5（反映注意力专注度）；
- **生理状态层**（hrv/ecg等）：hrv/bvp/ecg/resp，打分1-4（间接反映疲劳/应激，影响次之）。

#### 3. 同层细节适配
同层级内按细分重要性微调（如steering_angle > steering_velocity打2，hrv > bvp打4），既符合驾驶场景常识，又保证权重分配的合理性。
