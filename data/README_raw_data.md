# 数据集维度说明（能力计算17维，含路段类型总共18维）
本项目为**驾驶行为多模态数据集**，包含行为、眼动、生理三类信号，已完成时间对齐与预处理，维度及含义如下：

## 1. 行为信号 act（10 维，有效 9 维，60Hz）
| 维度索引 | 含义 |
|--------|------|
| 0      | 油门踏板开度 |
| 1      | 刹车踏板开度 |
| 2      | 方向盘转角 |
| 3      | 车速 |
| 4      | 纵向加速度 |
| 5      | 横向加速度 |
| 6      | 车辆 X 坐标 |
| 7      | 车辆 Y 坐标 |
| 8      | 相对理想路径偏移量 |
| 9      | 路段类型（施工区等）（能力计算时删除） |

## 2. 眼动信号 eye（3 维，120Hz）
| 维度索引 | 含义 |
|--------|------|
| 0      | 眨眼标准差（10s 窗口） |
| 1      | 注视点 X 均值 |
| 2      | 注视点 Y 均值 |

## 3. 生理信号 phy（5 维，100Hz）
| 维度索引 | 含义 |
|--------|------|
| 0      | 血容量脉搏 BVP |
| 1      | 心电信号 ECG |
| 2      | 呼吸信号 RESP |
| 3      | 心率 HR |
| 4      | 心率变异性 HRV |

---

## 数据预处理说明
- 采样率对齐：act(60Hz) / eye(120Hz) / phy(100Hz)
