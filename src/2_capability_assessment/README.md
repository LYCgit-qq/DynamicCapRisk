# 2_capability_assessment 模块介绍

本模块包含驾驶能力评估相关的核心脚本，主要用于计算和验证驾驶员的基准能力、能力波动和动态能力。

## 文件列表及功能

### baseline_capability.py
- **功能**: 计算基准驾驶能力 (Ab)
- **描述**: 使用K-means聚类算法对驾驶数据进行聚类分析，评估不同聚类数量的性能指标（轮廓系数、CH指数、DBI指数），确定最佳聚类数。基于聚类结果计算基准能力值，并生成相关的可视化图表（如PCA可视化、聚类指标图、能力分布图等）。
- **输入**: 预处理后的驾驶数据
- **输出**: 基准能力值CSV文件、聚类标签、可视化图表

### capability_fluctuation.py
- **功能**: 计算驾驶能力波动 (Afl)
- **描述**: 通过滑动窗口方法分析驾驶数据的时序波动，使用相关性分析和VIF（方差膨胀因子）筛选特征。采用AHP（层次分析法）和熵权法相结合的赋权方法计算综合波动指标。
- **输入**: 预处理后的驾驶数据（act/eye/phy）
- **输出**: 能力波动指标和赋权结果

### dynamic_capability.py
- **功能**: 计算动态驾驶能力 (Ad)
- **描述**: 综合基准能力和能力波动计算动态驾驶能力值。支持两种基准能力模式：Ab（基于聚类的群体基准）和Abc（个体化基准能力）。生成动态能力的计算结果和可视化。
- **输入**: 基准能力数据、能力波动数据
- **输出**: 动态能力值CSV文件、可视化图表

### capability_validator.py
- **功能**: 验证动态驾驶能力值 (Ad) 的有效性
- **描述**: 计算窗口级别的驾驶绩效指标，包括异常驾驶事件检测、车道保持稳定性分析、方向盘转角标准差（SASD）等。通过这些指标验证动态能力值的可靠性和有效性。
- **输入**: 原始驾驶数据和动态能力值
- **输出**: 绩效指标统计和验证报告

## 使用流程
1. 首先运行 `baseline_capability.py` 计算基准能力
2. 然后运行 `capability_fluctuation.py` 计算能力波动
3. 使用 `dynamic_capability.py` 结合上述结果计算动态能力
4. 最后用 `capability_validator.py` 验证动态能力的有效性

## 配置
所有脚本都支持YAML配置文件，位于 `config/` 目录下。可以通过命令行参数覆盖配置项。
